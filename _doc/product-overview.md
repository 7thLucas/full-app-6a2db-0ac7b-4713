# KYYXBOT — Product Overview

## Product Name
KYYXBOT

## Status
MVP scoped — ready to build.

## What It Is
A private AI chatbot application for a small, closed circle of friends. KYYXBOT gives the owner and their invited friends access to a capable AI assistant — instant answers on any topic and real hands-on coding help — in a clean conversational interface. It is never public-facing; access is invite-only.

## Problem It Solves
Generic AI tools are public by default — anyone can sign up and use them. There is no straightforward option for a private group that wants a shared AI for everyday questions and coding help without paying for enterprise tools or leaving access open to strangers.

## Target Users
**Primary**: The app owner and their close circle of friends — a small, private group.
Access is by invitation only. No public registration, no open access.

## Core Value / Day-One Feature
**AI Chat & Coding Assistant** — users can ask KYYXBOT anything (general knowledge, advice, explanations) and get real coding help (write code, debug, explain snippets). Fast, conversational, always available.

## Brand & Tone
- **Visual**: Dark/tech aesthetic — modern, sleek, bot-themed. Clean geometric icon, futuristic and exclusive feel.
- **Tone**: Casual and direct. No corporate language; the circle communicates informally.
- **Color palette**: Indigo/purple primary (established by Designer; logo live at platform).

## Strategic Principles
1. **Private by design** — no public registration, no open access. The circle stays closed.
2. **Conversation-first** — every interaction is a chat, never a form or menu.
3. **Genuinely capable** — the AI answers honestly and helps with real coding tasks, not surface-level responses.
4. **Expandable** — MVP ships the core Q&A + coding features; capabilities like image generation, file uploads, and integrations can be added as the circle's needs grow.
